# BurgerkingWebCloneCoding

<h3>프로젝트 소개</h3>

* 팀프로젝트인 버거킹 홈페이지 클론 코딩입니다.

<h3>사용 기술</h3>

* html, jsp, css, javascript, java, oracle xe 11gr2

<h3>기능 및 역할</h3>

* 애니메이션을 활용하여 header UI 및 기능 구현을 하였습니다.

* 사용자 이벤트 상태(진행중, 완료 등)에 따른 로직 처리를 하였습니다.

* 전체적인 서비스 UI 조정 및 구현을 하였습니다.

<h3>라이선스</h3>

* 버거킹 홈페이지를 활용하였습니다.
